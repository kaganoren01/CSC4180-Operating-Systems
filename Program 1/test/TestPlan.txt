Right Triangles Program - Test Plan
====================================

Program Overview
----------------
Counts right triangles from a set of 2D points using direction vectors.
Multi-process version distributes work across child processes.

Files:
- Triangles.java: Single-process version
- ProcessTriangles.java: Multi-process coordinator
- SingleProcessTriangleCounter.java: Child process worker
- TrianglesUtils.java: Shared algorithm (O(n^2))

Exit Codes: 1 = usage error, 2 = file error, 3 = format error

Running Tests
-------------

Compile:
  javac com/tryright/*.java

Single-process:
  java com.tryright.Triangles test/<testfile>.txt

Multi-process:
  java com.tryright.ProcessTriangles test/<testfile>.txt <num_processes>

Example:
  java com.tryright.ProcessTriangles test/test_long_list.txt 8


Performance Results
-------------------
Tested on 5000 points with varying process counts:

  Processes | Time (sec) | Speedup
  ----------|------------|--------
      1     |   16.69    |  1.00x
      2     |    7.48    |  2.23x
      4     |    6.51    |  2.56x
      8     |    6.87    |  2.43x
     16     |    7.66    |  2.18x

Optimal: 4 processes (matches CPU core count)
